# Math

## Expressoes matematicas padroes do java que nao sao as aritmeticas padroes:
|     Funcao      |      O que faz?      | 
|:---------------:|:--------------------:|
|  Math.sqrt(x);  |  Raiz quadrada de x  |    
| Math.pow(x, y); | X sera elevado por Y |   
|   Math.abs(x)   | Valor absoluto de X  |
